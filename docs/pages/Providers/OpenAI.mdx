# OpenAI 

## GPT-3

GPT-3 is a language model that uses deep learning to produce human-like text. It was released by OpenAI in July 2020. It is the successor to GPT-2, which was released in 2019. GPT-3 is the largest language model ever released, with 175 billion parameters. It was trained on a dataset of 45 GB of text, which is 10 times larger than the dataset used to train GPT-2. GPT-3 is capable of performing a wide range of tasks, including text completion, translation, summarization, and question answering. It can also be used to generate code, images, and music. GPT-3 is available for free to anyone with an API key.

## Chat GPT

Chat GPT is a chatbot that uses GPT-3.5 to generate responses. It was created by [OpenAI](https://openai.com/). It is available for free on the web. It can be used to chat with friends, or to test GPT-3. It is also available as an API.
