# identifier

The identifier property represents any kind of identifier for any kind of <Link href="/Thing">Thing</Link>, such as ISBNs, GTIN codes, UUIDs etc. Schema.org provides dedicated properties for representing many of these, either as textual strings or as URL (URI) links. See <a href="/docs/datamodel.html#identifierBg">background notes</a> for more details.
