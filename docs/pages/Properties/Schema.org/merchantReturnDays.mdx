# merchantReturnDays

Specifies either a fixed return date or the number of days (from the delivery date) that a product can be returned. Used when the <Link href="/returnPolicyCategory">returnPolicyCategory</Link> property is specified as <a class="localLink" href="/MerchantReturnFiniteReturnWindow">MerchantReturnFiniteReturnWindow</a>.
