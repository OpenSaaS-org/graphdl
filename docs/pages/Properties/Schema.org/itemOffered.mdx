# itemOffered

An item being offered (or demanded). The transactional nature of the offer or demand is documented using <Link href="/businessFunction">businessFunction</Link>, e.g. sell, lease etc. While several common expected types are listed explicitly in this definition, others can be used. Using a second type, such as Product or a subtype of Product, can clarify the nature of the offer.
