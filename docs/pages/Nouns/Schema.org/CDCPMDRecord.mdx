# CDCPMDRecord

A CDCPMDRecord is a data structure representing a record in a CDC tabular data format
      used for hospital data reporting. See <a href="/docs/cdc-covid.html">documentation</Link> for details, and the linked CDC materials for authoritative
      definitions used as the source here.
