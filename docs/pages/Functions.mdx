# ❪❫ Functions

GraphDL natively supports integrated functions.

Functions can be used for:

- Calculated Properties
- Default Values of Properties
- Function as a Property Type

## Calculated Properties

Calculated Properties are properties that are calculated based on other properties.

## Default Values of Properties

Default Values of Properties are values that are set when a new Resource is created.

## Function as a Property Type

Functions can be used as a Property Type. This allows you to define a property as a function.
