# ■●■ GraphDL - Graph Definition Language

GraphDL is a graph-based data language for describing data models and data relationships.

```yaml filename='graphdl.yaml' copy
🧑 Customer:
  has: [DeliveryAddress]
  places: [Order]
  liableFor: [Invoice]

🏘️ DeliveryAddress:
  receives: [Order]

🧾 Invoice:
  covers: [Order]

💸 Order:
  includes: [OrderItem]

🚥 ProductCategory:
  contains: [Product]

📦 Product:
  orderedIn: [OrderItem]
```

## ■ Noun

A Noun is a [Person](/Person), Place, Thing or Idea
